# Volley & Vibes — Urban Padel & Tennis Club Website

A premium, fully responsive single-page website built for a modern padel and tennis facility.

## 🎾 Live Demo
Open `index.html` in any modern web browser.

## ✨ Features

- **Dark Neon Aesthetic** — Electric lime on deep black for high-energy brand identity
- **Animated Hero Section** — Floating tennis balls with smooth CSS animations
- **Live Booking Form** — Court reservation system with validation
- **3-Tier Pricing Cards** — Psychology-optimized membership plans
- **WhatsApp Integration** — Floating chat button for instant customer contact
- **Mobile-First Responsive** — Perfect on phones, tablets, and desktops
- **Scroll Animations** — Intersection Observer powered fade-ins
- **Google Maps Ready** — Placeholder configured for real location embedding
